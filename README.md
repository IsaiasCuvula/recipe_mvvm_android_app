# MVVM RECIPE ANDROID APP Is an app where I show how to use MVVM, retrofit, dagger hilt, coroutine, liveData, kotlin, navigation component, and so on...

# Features

- Navigation Components
- Actions with Animations
- Safe Args
- MVVM
- Coroutine
- LiveData
- Retrofit 2
- Dagger-Hilt
